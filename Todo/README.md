#  React Todo App

A simple and interactive Todo List application built using **React**. This project was created while learning the fundamentals of React, including state management, event handling, list rendering, and component re-rendering.



##  Features

- ➕ Add new todos
- ✅ Delete completed todos
- 🚫 Prevent empty todo entries
- 🔄 Real-time input handling using controlled components
- 📋 Dynamically render todos using `map()`
- 🎨 Clean and responsive user interface

##  Getting Started

Clone the repository

```bash
git clone https://github.com/PRITIBHARDWAJ118/React-Learning.git
```

Navigate to the Todo project

```bash
cd React-Learning/todo
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
