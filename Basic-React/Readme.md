# 🚀 Basic React

This project is the first milestone in my **React.js Learning Journey**. It focuses on understanding the core building blocks of React and developing a strong foundation for building modern, component-based web applications.

The project includes hands-on implementations of React fundamentals, covering component creation, JSX, props, conditional rendering, styling, and event handling.

---

## 📖 Learning Objectives

- Understand React's component-based architecture.
- Build reusable and modular UI components.
- Learn JSX syntax and embed JavaScript within UI.
- Pass data between components using props.
- Handle user interactions with React events.
- Apply conditional rendering techniques.
- Organize React applications following best practices.

---

## 📚 Topics Covered

| Topic | Description |
|-------|-------------|
| **React Components** | Learned how to build reusable UI using functional components. |
| **Import & Export** | Organized React applications by importing and exporting modules. |
| **Named Exports** | Used named exports to export and import multiple values from a module. |
| **JSX (JavaScript XML)** | Wrote HTML-like syntax inside JavaScript to create dynamic user interfaces. |
| **Writing Markup with JSX** | Built UI layouts using JSX and embedded JavaScript expressions. |
| **React Fragments** | Used fragments to group multiple elements without adding unnecessary DOM nodes. |
| **Component Structure** | Organized applications into small, reusable, and maintainable components. |
| **Styling Components** | Styled React components using external CSS files and class names. |
| **Vite Development Environment** | Used Vite for fast development, hot module replacement, and optimized builds. |
| **React Props** | Passed data from parent to child components to create reusable and dynamic UIs. |
| **Conditional Rendering** | Displayed different UI elements based on conditions using JavaScript expressions. |
| **React Events** | Handled user interactions through React's event system. |
| **Click Events** | Implemented button click functionality using `onClick`. |
| **Non-Click Events** | Worked with events such as `onChange`, `onMouseOver`, and `onKeyDown`. |
| **Event Object** | Utilized the event object to access information about user interactions and event targets. |
